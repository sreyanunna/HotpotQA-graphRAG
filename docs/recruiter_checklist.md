# Recruiter Readiness Checklist

- README explains the problem, architecture, and quickstart within the first screen.
- Notebook is named descriptively and stored under `notebooks/`.
- Code is reusable from `src/` and runnable from `scripts/`.
- API keys are loaded from environment variables only.
- No Colab Drive paths are required for the public workflow.
- Data and result expectations are documented.
- Accuracy claims are separated from runtime-completion claims.
