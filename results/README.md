# Results

Generated files are written here by the batch runner.

- `batch_results.csv`: output from `scripts/run_batch.py`.
- `subgraphs/`: local Turtle and JSON subgraphs created during runs.
- `batch_results_sample.csv`: small shape example copied from visible notebook outputs.

For portfolio claims, add a `manual_verdict` column after checking each answer against ground truth. A successful pipeline run means the code completed; it does not automatically prove answer correctness.
